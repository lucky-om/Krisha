A bespoke, atmospheric web experience crafted with a modern "dark-gold" aesthetic. This project features interactive particle systems, smooth CSS animations, and a nostalgic Polaroid-style reveal.

This isn't just code; it's a digital gift. Built for Autumn—my favorite person and best friend forever.

🚀 Live Experience
View the Site: krisha.luckyverse.tech/flower/

How to Interact:
Mouse Move: Watch the deep-space background and stars shift with parallax.

The Golden Core: Click the central 🌻 (Sunflower) to trigger the transition and petal rain.

3D Polaroid: Hover over the image to see a realistic perspective tilt.

✨ Key Features
🌌 Dual Canvas Engines
Stars & Shooting Stars: Randomly generated twinkling stars with periodic high-speed shooting star animations.

Fireflies: 20+ floating particles using organic sine-wave movement and soft glows.

🖱️ Micro-Interactions
Custom Cursor: A gold-tinted cursor with a fluid trailing lag effect.

Click-to-Sparkle: Interactive 🌻 and ✨ emoji bursts on every click.

Petal Rain: A celebratory waterfall of flowers and sparkles upon entry.

🎵 Audio Synthesis
Web Audio API: Generates a custom synthetic chime sequence in real-time upon entry—no external MP3 files required.
